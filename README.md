# 2D-Vertical-ShootingGame
2D 종스크롤 슈팅게임 만들기

총알발사, 보스 패턴(원형, 부채꼴, 난사 등), 피격 및 무적(투명), 총알 및 적 비행기 오브젝트 풀링, 이동 및 슈팅 UI, 아이템 드랍 및 레벨업과 동시에 데미지 총알수 다르게 구현,
필살기, 맵 스크롤링을 통해서 원근감있는 맵 구현, 적 리스폰 텍스트 파일로 관리, 모바일 빌드까지 성공!
